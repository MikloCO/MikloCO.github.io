Newest Project

Playable Tetris on a gameboy 1989 with Three.js

I've modelled the nintendo gameboy 1989 in Houdini. I textured it in substance painter and then exported it as .glb to use on the web. With three.js and webgl, I was able to create a playable version of tetris that runs smoothly in the browser. The props in the render are created with blender.


A link to the gameboy will be added soon.
