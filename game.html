Nintendo Gameboy

Home page Nintendo Gameboy

Controls

  • (Pan) alt + ctrl + mouse click
  • (Rotate) mouse click + drag
  • (Zoom) scroll
  • (Move tetris vertically) ← →
  • (Move tetris horizontally) ↑ ↓
  • It also works to click physical arrows

Info

  • Modelled in Houdini
  • Textured in Substance Painter
  • Animated in Blender
  • Exported fr Blender as .glb
  • Rendered with Three.js
  • Tetris blocks are procedurally drawn using glsl (shader)
  • Game state is passed to shader via uniforms
  • Core game logic and state are managed in JavaScript
